Data Transformation 12

In this step, you will clean and transform the data:

  1. Select Transform Data from the Home tab.
  2. Check if the dataset has any inconsistencies, such as:
    • Inconsistent formats (e.g., date formats like MM-DD-YYYY and YYYY/MM/DD).
    • Null values or missing data in critical fields.
    • Duplicate records or other data quality issues.
  3. If no inconsistencies are found, confirm that the data is clean and consistent.
    • The data was clean and did not contain any inconsistencies.
  4. Review the data to ensure accuracy across all columns.
  5. Click on File and select Close and Apply to load the data into Power BI.
