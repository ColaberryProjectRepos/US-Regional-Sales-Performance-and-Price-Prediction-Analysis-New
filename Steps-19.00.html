Adding Date Slicer 19



1.  In the Report View, open the visualization pane, then Select Slicer from visualization

2.  In the fields, open ‘Calendar’ Table, drag the Date.

    a. In the Formatting Tab-->Visual-->Values: Font = Segoe UI , Size = 9, Slider: Off

    b. In the Formatting Tab-->General-->Properties-->Size: Height = 100, Width = 227, Position: Horizontal = 946, Vertical = 0, Title: Date, Font = Segoe UI Bold, Size =14, Horizontal alignment= Center.
