Adding Project title, Company Logo and Sales logo 18


1.   In the report view, Select Insert ribbon>Text box 

     a.   Type the Project Name ‘US Regional Sales Performance and Price Prediction Analysis’ 

     b.   Highlight the Text and adjust Size = 18, Font = Segoe (Bold), Center 

     c.    In the Formatting Tab-->General-->Properties-->Size: Height= 61, Width = 702,              Position: Horizontal = 232, Vertical = 0, Effect: Background: Off

2.   In the Report View, Select Insert ribbon > Image > and select the Company logo. 

     a.    In the Formatting Tab-->General--> Properties-->Size: Height = 61, Width = 190, Position: Horizontal = 0, Vertical = 0, Background: Off. 

3.   Repeat the same step 2 for Sales Logo 

      a.  In the Formatting Tab-->General--> Properties-->Size: Height = 61, Width =87, Position: Horizontal = 1193, Vertical = 0, Background: Off.
