Creating Scatter Chart – Store Performance by Discount vs Revenue

1.      Click on an empty space on the canvas, and from the visualizations pane select Scatter chart

2.      Fill up figures provided below for the necessary fields

 

Build Visual Pane:

·        Values  = StoreID

·        X axis= Total Revenue

·        Y-axis = Discount Applied Percentage

·        Size = Order Qty

Format Visual Pane:

Visual:

1.In the Formatting Tab-->General-->Properties-->Size: Height=275, Width=552, Horizontal=722, Vertical=100

Background =On; Transparency = 100%

Visual border =Off

 Title=On, Text=Store Performance by Discount vs Revenue, Font=Segoe UI (12), Text color=White, Background color=#005F73, Horizontal alignment=Center

2.In the Formatting Tab-->visual-->X-axis: Values = On, Font=Segoe UI 9; Color=#605E5C;  Title=On, Segoe UI 10; Color=#252423

Y-axis: Range: Auto, Values=On, Font=Segoe UI 9; Color=#73361C; Title=On, Segoe UI 10; Color=#252423

Legend: on,  Text= Segoe UI 10; Color =#605E5C; Title=on; Title Text = Unit Price

Markers: Categories=All;  Color by category =Off, ; Color fx, Default Color-Categories , Minimum :#7BDEE1, Maximum :#005f73

Category Label: on; Values - Font=Segoe UI 9; Color=#605E5C
