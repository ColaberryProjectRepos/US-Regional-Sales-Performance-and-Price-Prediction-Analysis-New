Adding KPI Cards and Refresh Date 20



- Creating Total Revenue KPI

1. Click on an empty space on the canvas, and from the Build pane select Card   visual

2. On the build pane drag and drop the Total Revenue measure on the fields format it to decimal number 1.

 

visual Format

In the Formatting Tab-->Visual-->Callout value: Font = Segoe UI , size = 12, Category label: Off

In the Formatting Tab-->General-->Properties-->Size: Height = 92, Width = 170, Position: Horizontal = 0, Vertical = 99, Title: Total Revenue, Font = Segoe UI , Size = 12, Color = White, Background Color= #005F73, Center, Effect: Background: On, Color = White, Transparency = 100

 


- Creating Discount KPI

1. Copy Total Total Sales KPI by pressing CTRL C and paste it by CTRL V

2. On the build pane drag and drop the Total Discount Applied measure from the MeasureTable on the fields (delete the Total Revenue field). Format the value to 1 decimals.

 

visual Format

In the Formatting Tab-->Visual-->Callout value: Font = Segoe UI , size = 12, Category label: Off

Creating  Refresh Date

1.   Click on an empty space on the canvas, and from the visualizations pane select Visual Card.

2.   Drag the Refreshed Date from Refresh Date Table to the fields value box.

3.   In the Formatting Tab-->Visual-->Callout value: Font = Segoe UI , size = 10, Color=Black , Category label: Off
