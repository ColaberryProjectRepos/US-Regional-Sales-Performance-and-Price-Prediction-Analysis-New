Creating Sales Revenue by Year/ Channel 22



-Creating Total Revenue by Year  Line chart

1.      Click on an empty space on the canvas, and from the visualizations pane select Line Chart

2.      Fill up figures provided below for the necessary fields

Build Visual Pane:

·        X axis = Year

·        Column y -axis= Total Rev

·        Line y-axis=Rev YoY%, Dis YoY%,Unit YoY%,OQty YoY%

·        Tooltips= Rev YoY%, Dis YoY%,Unit YoY%,OQty YoY%

 

Format Visual Pane:

Visual:

1.     In the Formatting Tab-->x-axis: On, Font= Segoe UI , size= 9, Color = Black. Maximum height 25%, Layout 20px Title: Off.   Y-axis: On, Font = Segoe UI , size= 9, Color= Black, Display units=Millions. Title: Off.      Secondary y-axis: On, Align zeros: Off, Font= Segoe UI , size= 9, Color = Black. Title: Off.  Legend: On, Position = Top left, Style: Marker Text: Font= Segoe UI , size= 10, Color = Black. Title: On.    Gridlines: Horizontal: On style = Solid,  width = 1  

Columns: Categories: All Colors=#004C5D.Transparency  0%                                                             Line Series=All, Line style= Solid, Jointype: Round, with 3px ,type Linear Data labels: Off , shade area:Off, series labels :Off

 

2.     In the Formatting Tab-->General-->Properties-->Size: Height = 275, Width = 552, Position: Horizontal = 169, Vertical = 100, Title: Total Revenue by Year, Font = Segoe UI , Size = 12, Color = White, Background: #005F73, Center, Effect: Background: On, Color = White, Transparency=100%. 

       Tooltips: On, Text Font = Segoe UI , Size = 10, Label Color = Black, Value Color = Black and     Background: Color=White
